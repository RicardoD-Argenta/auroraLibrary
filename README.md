## 📖 Sobre o projeto

O **Aurora Library** é um sistema web para gerenciamento de bibliotecas escolares, desenvolvido com o objetivo de ser uma solução gratuita e acessível para escolas públicas e privadas.

O projeto nasceu de uma iniciativa pessoal de **Ricardo**, que queria resolver um problema real: ajudar a própria escola onde estudava a organizar e controlar o acervo da biblioteca de forma eficiente. O que começou como um projeto pessoal cresceu e ganhou forma para se tornar algo maior — uma ferramenta que qualquer escola possa usar.

## 🌍 Open Source

O Aurora Library é um projeto **open source**, desenvolvido de forma aberta e colaborativa. O objetivo é que ele seja disponibilizado gratuitamente para escolas, sem custo de licença.

Se você quiser contribuir, fique à vontade para abrir issues e pull requests!

## ⚠️ Status: Beta

O projeto ainda está em fase **beta**. Funcionalidades podem mudar, bugs ainda existem e há um bom caminho pela frente. Toda contribuição e feedback são muito bem-vindos.
